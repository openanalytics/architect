\textbf{СКЛАДОПОДЎЛ}

\indent
Для укра∙нсько∙ мови, як ў для ўнших слов'янських\linebreak 
мов, характерне загальне тяжўння до вўдкритих складўв.\linebreak
Це виявляїться в тому, що межа мўж окремими складами\linebreak
проходить найчастўше пўсля голосного звука перед при-\linebreak
голосним, наприклад: \emph{се-стра, мў-ста, за-здрий.} Проте\linebreak
при збўговў кўлькох приголосних складова межа часто\linebreak
проходить всерединў цўї∙ групи, наприклад: \emph{книж-ка,\linebreak 
гол-ка, дзвўн-кий.}


Коли голосний поїднуїться з одним приголосним,\linebreak 
подўл слова на склади труднощўв не викликаї.  При збў-\linebreak
говў кўлькох приголосних встановити складову межу\linebreak 
буваї важко. Ўснують такў основнў правила складопо-\linebreak
дўлу:

1.~Один приголосний, що сто∙ть мўж голосними, зав-\linebreak
жди належить до наступного складу, наприклад: \emph{по-\linebreak
су-ха, за-го-ро-да, си-то.}

2.~Два шумнў приголоснў (обидва дзвўнкў або обидва\linebreak
глухў) належать звичайно до наступного складу, на-\linebreak
приклад: \emph{мў-сто, во-ско-вий, дў-жда-ти-ся, не-спо-кўй,\linebreak
при-∙-зди-ти, не-ща-стя.}

3.~Три приголоснў, коли два першў з них шумнў\linebreak
(обидва дзвўнкў або обидва глухў), а третўй сонорний, на-\linebreak
лежать до наступного складу, наприклад: \emph{го-стрий,\linebreak
по-стрўл, за-здро-щў.} 

4.~Сусўднў приголоснў, перший з яких бўльш гучний,\linebreak
нўж другий, належать до рўзних складўв, наприклад:\linebreak
\emph{каз-ка, груд-ка, вез-ти.}

5.~Якщо мўж голосними ї два або бўльше приголос-\linebreak
них, то звуки [\emph{j}], [\emph{в}], [\emph{р}], 
[\emph{л}], [\emph{м}], [\emph{н}], що йдуть пўсля\linebreak
голосного, належать до попереднього складу, а звуки,\linebreak
що стоять пўсля них,--- до наступного: \emph{гай-ка, май-\linebreak
струвати, гав-кати, гўр-ко, пал-ко, ям-ка, син-ку.}


6.~Коли другим приголосним ї звук [\emph{j}], [\emph{в}], [\emph{р}],\linebreak{} 
[\emph{л}], [\emph{м}] або [\emph{н}], то разом з попереднўм вўн вўдходить\linebreak\eject\noindent 
до наступного складу: \emph{за-б/йу/, лю-блю, му-дрий, ну-\linebreak
дний, по-свист, Ку-зьма.}
